## WORK UTE / VEHICLE

### What You Need
- **Dual cab ute or van** — needs to fit: yourself + 1 apprentice/runner + tools + ladder + recovery bottle
- **Ladder rack or roofracks** — for ladder transport
- **Tow bar** — useful for trailer if needed

### Recommended Setup
| Item | Budget Option | Quality Option |
|------|--------------|----------------|
| Work ute | Toyota Hilux ($35-50k used) | Ford Ranger ($40-55k used) |
| Ladder racks | Rhino-Rack or DIY pipe rack | Off-road rack with lockable box |
| Van (alternative) | Toyota HiAce ($25-40k used) | Mercedes Sprinter ($50k+) |

### Vehicle Must-Haves
- [ ] Canopy or hard lid (protects tools)
- [ ] Thule or similar roof rack for ladder
- [ ] 12V socket in cabin (for phone charger)
- [ ] Rubber matting in tray (protect gear)
- [ ] Tow bar + rated recovery point

### Startup Vehicle Cost
- Second-hand ute in good condition: **$30,000–$50,000**
- New work vehicle: **$45,000–$65,000**
- Finance option: many banks do chattel mortgage for ABN holders

## SECONDHAND vs NEW

### Buy Secondhand (Fine):
- Hand tools (Stanley, Knipex, etc.)
- Manifold gauges (check calibration)
- Multimeter
- Pipe cutters
- Flaring tools
- Brazing kit
- Vacuum pump (check oil, test vacuum)
- Vehicle (get mechanic check)

### Buy New (Worth It):
- Recovery machine (safety + legal compliance)
- Leak detectors (accuracy matters)
- Insulation testers (legal requirement)
- Capacitors (they're cheap new)
- Safety gear

## INSURANCE — TOOL COVER

**Get tool insurance separate from vehicle insurance.**

- **Tool coverage:** $10,000–$20,000 riders on contents insurance
- **Don't leave tools in the van overnight** if possible
- **Lock the canopy/van** — use a deadlock
